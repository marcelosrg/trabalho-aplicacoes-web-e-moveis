Sobre mim

Olá, meu nome é Marcelo. Iniciei minha trajetória no desenvolvimento web em 2022 como desenvolvedor frontend em uma agência de marketing, onde era responsável pela criação de sites institucionais, landing pages e outros tipos de páginas web. No dia a dia, utilizava tecnologias como HTML, CSS, JavaScript, Bootstrap, entre outras ferramentas voltadas para o desenvolvimento frontend. Com o tempo, comecei a aprofundar meus estudos em desenvolvimento de APIs e, atualmente, meu foco está na parte de servidor (BackEnd), utilizando tecnologias como Java, Spring Boot e bancos de dados SQL e NoSQL. Atualmente, trabalho como analista de dados em um centro universitário, onde atuo na elaboração de relatórios no sistema TOTVs RM, utilizando SQL Server para consultas no banco de dados da instituição.

Experiência

Atualmente trabalho como Analista de Dados em um centro universitário, onde sou responsável pela elaboração de relatórios no sistema TOTVs RM, utilizando SQL Server para consultas no banco de dados da instituição. Nesta função, extraio e analiso dados para fornecer insights e apoiar a tomada de decisões. Iniciei minha carreira profissional em 2022 como Desenvolvedor Frontend em uma agência de marketing. Nesta posição, fui responsável pela criação de sites institucionais, landing pages e outras interfaces web, utilizando tecnologias como HTML, CSS, JavaScript e Bootstrap. Durante minha trajetória profissional, venho aprofundando meus conhecimentos em desenvolvimento backend, com foco em Java, Spring Boot e bancos de dados SQL e NoSQL, expandindo assim minhas competências para além do desenvolvimento frontend.

Formação Acadêmica

Atualmente estou cursando Engenharia de Software, onde desenvolvo uma sólida base teórica em algoritmos, estruturas de dados, arquitetura de software e metodologias ágeis. Paralelamente à graduação, invisto em minha especialização técnica através de um curso focado em desenvolvimento backend com Java, aprofundando conhecimentos em Spring Boot, APIs RESTful, bancos de dados relacionais e não-relacionais. Esta combinação de formação acadêmica com capacitação técnica específica me proporciona tanto o entendimento dos fundamentos da engenharia de software quanto as habilidades práticas para implementação de soluções robustas e escaláveis no mercado de trabalho.
